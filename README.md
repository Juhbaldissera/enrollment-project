# enrollment-project
Project developed on Brana's course: https://app.branas.io/public/products/cd4aea57-4b43-41e9-bda5-f487c3817b29. 

## Requirements
- Node 12

## Installing
```bash
$ git clone git@github.com:Juhbaldissera/enrollment-project.git
$ cd enrollment-project
$ npm i
```

## Testing
```bash
$ npm run test
```

## Running
```bash
$ npm run build
$ node dist/index.js
```
